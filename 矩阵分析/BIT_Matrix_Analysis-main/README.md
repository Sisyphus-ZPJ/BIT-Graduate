# 北京理工大学研究生《矩阵分析》资料分享
北理工《矩阵分析》这节课有的老师给往年试卷，有的老师不给。很不巧我们老师就不给，不给就只能找人要。**本人对校内树洞某些人高价倒卖的行为非常不爽**，遂决定把自己搜集到的资料分享出来，也算是发挥一下程序员的开源精神。  
本仓库包括北京理工大学研究生《矩阵分析》课的课件、教材和历年试卷等内容。如果大家有补充的资料，也欢迎提交pull request，我会查看后合并。这样也可以方便以后需要这些资料的同学。
